## Auto Ward Lee Sin

> A simple python script which auto **spam W** for you so you just **press ward shortcut** to perform ward dash.

> Please place wards in 2 and 4.

> Install dependencies.

```shell
pip install pynput pystyle
```

## Running

```shell
python main.py
```
